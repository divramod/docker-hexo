# how to use


## clone your blog

```bash
git clone git@bitbucket.org:divramod/blog.git ~/blog

```

## build docker and npm install
```bash
cd
git clone git@github.com:divramod/docker-hexo.git
cd docker-hexo
sh scripts/build.sh
```

## run the blog

```bash
cd ~/docker-hexo
sh scripts/run.sh
```

## update the blog

```bash
cd ~/docker-hexo
sh scripts/update.sh
```
